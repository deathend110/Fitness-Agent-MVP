# 周期计划库的实现
目前的训练计划是没有“周期”这个概念的。后续要添加周期这个概念，周期长度可自拟，1周，2周还是多周，比如6周。上一周期进入下一周期，计划会依据当前的周期计划自动更新。同时支持开放的自定义周期计划，自己设定每个周期长度，每个周期的训练内容，每个周期的增长step等。具体实现这个功能时再分析。

# 预制计划库
实现1后，可以有预制计划库，里面是各种经典健身计划，选择后会依据你的三大项1RM自动覆盖训练计划表，增肌和减脂期计划比较自由，所以这里主要是各种力量计划
5x5，疯牛5x5，德州计划，condito6周计划，黑又壮计划，以及常见的线性周期容量递增计划
具体实现这个功能时再分析。
