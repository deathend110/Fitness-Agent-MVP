
# 1 AI对话框 
- enter直接发送消息, shift+enter才是换行
- 对话框设计应该类似主流大模型网页端的设计,比如claude或者GPT的设计. 把对话框放到中间而不是旁边，具体你可以参考.
- 未发送的prompt记忆功能，我可以打字打一半切去其他页，回来时保留有原来的记录
- 后台思考功能。现在的模型是必修停留在AI教练界面才会思考，我离开就不行了，需要支持后台思考，允许我离开页面
- 我需要能选择模型，从接入的API中列出可用模型（参考Gemini页面的设计，在提示词输入框的左边或者右边有可展开的选项）
- 模型输出是MD格式的文本. 应该要加载一个MD渲染器, 渲染MD格式, 而不是直接输出没有渲染的MD语法内容
- 支持文件上传解析主要是图片和excel, md文件和docx文件. 上传的文件保存在本地的一个缓存文件夹(不要C盘, 存在软件所在文件夹).
- 对话应该维持在最新一条，而不是每次切换，刷新页面后变成首条
- 具体布局可以参考：Astrbot 聊天页面.png
