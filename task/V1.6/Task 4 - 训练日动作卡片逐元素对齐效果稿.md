# Task 4 训练日动作卡片逐元素对齐效果稿

- **估时**：65 分钟
- **依赖**：Task 3
- **目标**：将训练日动作卡片的元素、层级、边框、标签、重量区和备注区严格对齐效果稿。
- **主要文件**：
  - `src/components/PlanExerciseItem.jsx`
  - `src/utils/planExerciseCard.js`
- **主要操作**：
  - 按效果稿重排标题、主辅项标签、1RM 来源说明、组次、重量、RPE、备注。
  - 保证默认外观与效果稿一致。
  - 继续兼容百分比动作、固定重量、自重、长动作名和空备注。
  - 保留右上角操作入口位，但默认外观必须先对齐效果稿。
- **完成信号**：
  - 训练日卡片默认展示高度接近效果稿。
  - 百分比 / 固定 / 自重动作都能落在效果稿结构里。
- **验证方式**：
  - 对照效果稿检查训练日卡片结构。
  - 用多种动作类型验证展示。
