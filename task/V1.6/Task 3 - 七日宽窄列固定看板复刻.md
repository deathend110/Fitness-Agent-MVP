# Task 3 七日宽窄列固定看板复刻

- **估时**：60 分钟
- **依赖**：Task 2
- **目标**：将训练计划页的 7 天布局改为效果稿中的固定展示看板，去掉训练日展开 / 收起逻辑。
- **主要文件**：
  - `src/tabs/PlanTab.jsx`
  - `src/components/plan-grid/*`
  - `src/utils/planLayout.js`
- **主要操作**：
  - 去掉训练日与休息日的展开 / 收起交互。
  - 7 天列需要能**自适应**实现展示效果稿中的宽窄比例布局。无论多少天训练多少天休息，其训练日和休息日的宽窄都是不一样的，除非全都是训练或者休息日。但是页面最终效果要能依据实际情况调整宽窄大小。宽窄比例维持这个比例。
  - 保持正常 16:9 视口下不横向滚动。
  - 为后续卡片真实数据映射保留稳定的列结构。
- **完成信号**：
  - 页面默认直接展示完整 7 天。
  - 训练日宽列、休息日窄列与效果稿一致。
  - 不再出现“展开”“收起”入口。
- **验证方式**：
  - 打开训练计划页，确认 7 天完整固定可见。
  - 检查桌面视口下不出现左右滚动。
