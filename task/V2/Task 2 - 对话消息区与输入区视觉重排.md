# Task 2 主对话区与输入卡片严格对齐示意图

- **估时**：25 分钟
- **依赖**：Task 1
- **目标**：按 `Astrbot 聊天页面.png` 与 `Astrbot 聊天页面1.png` 重新排布主对话区，让消息区、欢迎语和输入卡片严格对齐示意图。
- **主要文件**：
  - `src/components/CoachConversationPanel.jsx`
  - `src/components/CoachComposer.jsx`（如需要新增）
  - `src/components/CoachMessageList.jsx`（如需要新增）
  - `src/tabs/CoachTab.jsx`
- **主要操作**：
  - 把主区域重建为空状态与已对话状态两种显示形态。
  - 空状态下只保留欢迎语与中心输入卡片，不保留旧的说明卡、辅助卡或预览卡。
  - 已对话状态下消息流按示意图左右分布，输入卡片固定在底部。
  - 调整消息气泡、滚动区域、空态文案和输入占位，让它们贴合示意图。
  - 不改发送逻辑，不改草稿逻辑，不改任何数据结构。
- **完成信号**：
  - 空状态时只看到欢迎语和中心输入卡片。
  - 已对话状态时消息流、输入框和滚动关系与示意图一致。
  - 主区域不再残留旧的三栏工具面板感。
- **验证方式**：
  - 打开空状态页面，确认欢迎语与输入卡片位置和比例正确。
  - 发送一条消息后，确认主区域切换到已对话状态。
  - 检查长消息、空消息和发送中状态是否仍然保持示意图的层级。
