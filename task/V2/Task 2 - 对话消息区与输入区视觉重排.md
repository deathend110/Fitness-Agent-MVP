# Task 2 对话消息区与输入区视觉重排

- **估时**：25 分钟
- **依赖**：Task 1
- **目标**：只调整消息区与输入区的视觉结构、间距和层级，让它更接近主流聊天产品。
- **主要文件**：
  - `src/components/CoachConversationPanel.jsx`
  - `src/tabs/CoachTab.jsx`
- **主要操作**：
  - 重排消息气泡的宽度、边距、对齐方式和滚动区域。
  - 优化输入区与消息区的分隔关系。
  - 调整按钮、提示文案、空状态的视觉位置。
  - 不改发送逻辑，不改草稿逻辑，不改任何数据结构。
- **完成信号**：
  - 聊天消息区看起来更像网页端聊天界面。
  - 输入区在视觉上和消息区形成明确层次。
  - 页面没有多余的工具面板感。
- **验证方式**：
  - 打开 AI 教练页，确认消息区和输入区的视觉重心在中间。
  - 检查长消息、空消息和发送中状态的展示是否协调。
