# Task 9 我的档案页新UI适配

- **估时**：45 分钟
- **依赖**：Task 1
- **目标**：让“我的档案”页面在新壳层下同步迁移到效果稿风格，保持与训练计划页一致的布局语言、卡片层级和操作节奏。
- **主要文件**：
  - `src/tabs/ProfileTab.jsx`
  - `src/components/` 下档案页相关组件
  - `src/index.css`
- **主要操作**：
  - 重构档案页标题区、表单区和数据展示卡片，使其适配左侧壳层后的主内容节奏。
  - 统一卡片圆角、边框、留白、标题层级和辅助说明文风。
  - 让表单布局在桌面视口下更贴合新设计，避免保留旧版“独立工具面板”气质。
  - 保证档案关键信息和常用输入项在全屏桌面视口下可以完整看到，不靠横向滚动或过度折叠承载。
  - 保持现有导入导出、档案填写、1RM 填写等业务能力不变。
- **完成信号**：
  - 档案页视觉和布局明显属于同一套新 UI。
  - 表单、摘要卡片与按钮状态不再像旧页面拼接上来的内容。
  - 页面在桌面全屏下无横向滚动。
  - 档案主信息、1RM 输入与关键操作入口在桌面全屏下完整可见。
- **验证方式**：
  - 打开档案页，检查标题区、表单区和数据区是否统一。
  - 填写档案并刷新，确认功能未回归。
  - 检查桌面全屏视口下无左右滑动。
  - 检查主要档案字段是否无需额外横向移动即可阅读和编辑。
