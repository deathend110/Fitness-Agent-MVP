# Task 11 AI 教练页新UI适配（不含 V2 能力增强）

- **估时**：50 分钟
- **依赖**：Task 1
- **目标**：在不提前实现 `V2` AI 对话框增强功能的前提下，让当前 AI 教练页先同步迁移到新 UI 壳层与页面布局风格。
- **主要文件**：
  - `src/tabs/CoachTab.jsx`
  - `src/components/CoachConversationPanel.jsx`
  - `src/components/PromptPreviewPanel.jsx`
  - `src/components/AdoptCard.jsx`
- **主要操作**：
  - 重构 AI 教练页的页面骨架、信息区块和卡片层级，使其与新壳层协调。
  - 保留当前 V1 已有能力：聊天、上下文预览、采纳卡片、错误提示、配置状态。
  - 不在本任务中实现 V2 才处理的对话框深度重构、DeepSeek API 优化或 Agent 增强。
  - 调整页面布局，使当前内容在桌面全屏视口下无横向滚动。
  - 保证聊天主区、上下文预览、采纳卡片和状态信息在桌面全屏下完整可见，而不是彼此挤压到不可读。
- **完成信号**：
  - AI 教练页不再停留在旧版视觉层级，能与新应用壳层统一。
  - 当前 AI 相关功能仍可正常使用。
  - 页面在全屏桌面视口下无左右滑动。
  - 聊天区和辅助信息区都保持可读，不因压缩而失去主要内容。
- **验证方式**：
  - 打开 AI 教练页，确认配置区、聊天区、预览区、采纳区层级清楚。
  - 发送一次消息并检查当前功能未回归。
  - 检查桌面全屏视口下无横向滚动。
  - 检查聊天主内容和辅助区块是否都能完整浏览。
