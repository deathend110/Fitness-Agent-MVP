# Task 1 应用壳层与侧边栏导航重构

- **估时**：45 分钟
- **依赖**：无
- **目标**：把应用整体壳层从当前的顶部导航布局迁移到效果稿对应的“左侧侧栏 + 主内容区”结构，同时保证四个主页面仍可正常切换，并为其余 tab 的新风格迁移打底。
- **主要文件**：
  - `src/App.jsx`
  - `src/index.css`
  - `src/components/` 下新增壳层或导航组件
- **主要操作**：
  - 重构应用顶层布局，加入固定左侧导航区、主内容区和底部状态区。
  - 让“我的档案 / 训练计划 / 今日日志 / AI 教练”迁移到侧栏导航。
  - 保留当前 tab 切换逻辑，但外观、布局、层级改成效果稿风格。
  - 增加底部状态区的 MVP 版本，包括本地存储状态提示与快捷按钮位。
  - 约束主内容区宽度与滚动策略，为后续训练计划页“16:9 全屏无横向滚动”提供基础。
  - 只做壳层承载，不在本任务里重写各 tab 内部业务。
- **完成信号**：
  - 应用整体布局已经接近效果稿的侧栏结构。
  - 四个主页面仍可切换，且内容区不被壳层改坏。
  - 当前蓝紫高亮 + 白色底板风格与新壳层统一。
  - 主内容区滚动策略明确，不会因为新壳层默认制造整页横向滚动。
- **验证方式**：
  - 打开首页，确认左侧存在固定导航。
  - 逐个点击四个导航项，确认页面内容正常切换。
  - 检查桌面端宽屏下侧栏与内容区比例是否稳定。
