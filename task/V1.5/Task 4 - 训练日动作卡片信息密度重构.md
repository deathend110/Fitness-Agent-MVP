# Task 4 训练日动作卡片信息密度重构

- **估时**：60 分钟
- **依赖**：Task 3
- **目标**：把训练日列中的动作卡片升级为效果稿中的高信息密度卡片样式，更清晰地表达主辅项、重量、次数、RPE 和备注。
- **主要文件**：
  - `src/components/PlanExerciseItem.jsx`
  - `src/components/PlanDayCard.jsx`
  - `src/utils/planExerciseCard.js`
- **主要操作**：
  - 重构动作卡片排版，让动作名、主辅标签、负重来源、组次、重量显示更清楚。
  - 强化主项 / 辅项视觉差异，但仍沿用现有数据结构。
  - 优化长动作名、空备注、自重动作、固定重量动作、百分比动作的展示细节。
  - 在不依赖横向滚动的前提下重新设计信息密度，保证关键字段尽量都能在卡片主视图中看到。
  - 为后续操作菜单保留稳定入口位。
- **完成信号**：
  - 单个训练日内多个动作卡片更容易快速扫描。
  - 百分比重量与实际重量表达直观。
  - 自重、空备注、长动作名等边界情况不破版。
  - 关键字段不会因为压缩布局而被大量隐藏或只剩下模糊缩写。
- **验证方式**：
  - 使用含主项、辅项、固定 kg、自重动作的计划数据查看卡片表现。
  - 检查动作名、标签、重量和备注在窄列下是否溢出。
  - 检查在 16:9 全屏视口下，多列同时存在时信息是否仍可完整阅读。
