### Task 0.1 初始化 Vite React 项目

- **估时**：30 分钟
- **依赖**：无
- **目标**：创建可运行的 React 前端项目。
- **主要操作**：
  - 使用 Vite 初始化 React 项目。
  - 安装 Tailwind CSS。
  - 配置 `.gitignore`，确保 `.env` 不被提交。
  - 创建基础目录：`src/tabs`、`src/components`、`src/utils`、`src/api`。
- **完成信号**：
  - `npm install` 成功。
  - `npm run dev` 能启动本地页面。
  - 页面显示 RepMind 初始界面。
- **验证方式**：
  - 浏览器打开本地地址，确认无白屏。
  - 检查 `.env` 已被 `.gitignore` 忽略。
