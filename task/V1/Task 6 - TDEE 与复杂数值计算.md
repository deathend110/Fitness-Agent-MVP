# Task 6 TDEE 与复杂数值计算

- **估时**：40 分钟
- **依赖**：Task 5
- **目标**：补齐 TDEE、体脂等复杂指标的计算与展示，并引入大模型辅助推算，让模型结合已有身体数据、训练记录和饮食输入一起判断复杂数值，提升结果的合理性与可解释性。
- **主要文件**：
  - `src/utils/calc.js`
  - `src/utils/prompt.js`
  - `src/tabs/TodayTab.jsx`
  - `src/components/PromptPreviewPanel.jsx`
  - `tests/`
- **主要操作**：
  - 先整理可用的身体数据、训练记录和饮食记录。
  - 再定义本地可确定的基础计算口径，例如 BMR、训练消耗、TDEE 初算等。
  - 把这些结构化数据注入 prompt，让大模型在此基础上辅助推算更复杂的指标。
  - 将模型推算结果和本地计算结果一起展示到页面中，确保用户能看到完整依据。
  - 保证 prompt 预览和实际发送给 AI 的上下文保持一致。
- **完成信号**：
  - 复杂指标能稳定计算，并且会结合大模型辅助推算。
  - 页面展示和 prompt 注入保持一致。
  - AI 能基于身体数据、训练输入和饮食数据给出更可靠的复杂指标判断。
- **验证方式**：
  - 使用固定档案、训练和饮食数据，确认结果数值稳定且可解释。
  - 检查页面上的复杂指标是否与上下文预览中的内容一致。
