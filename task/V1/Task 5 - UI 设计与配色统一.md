# Task 5 UI 设计与配色统一

- **估时**：25 分钟
- **依赖**：Task 4
- **目标**：统一 RepMind 的视觉风格，重点把整体配色调整为“清亮的蓝紫色高亮 + 白色背景底板”的方向，尽量贴近 Codex 桌面 app 那种干净、明亮、层级清晰的观感。
- **主要文件**：
  - `src/index.css`
  - `src/App.jsx`
  - `src/components/*.jsx`
  - `src/tabs/*.jsx`
- **配色要求**：
  - 整体底板以白色或接近白色为主，保持轻盈、干净的产品气质。
  - 重点状态、强调信息、选中态和关键按钮使用清亮的蓝紫色高亮。
  - 次级信息使用更浅的灰蓝或冷灰色，保证层级清楚。
  - 避免过重、过暗、过脏的背景色，让页面保持明快。
  - 视觉气质参考 Codex 桌面 app：克制、清晰、专业，但有明显的蓝紫高亮识别度。
- **主要操作**：
  - 统一背景、边框、按钮和卡片的颜色系统。
  - 让白色背景底板成为页面主视觉，蓝紫色只负责强调、选中和引导视线。
  - 调整页面层级、留白和卡片对比度，避免界面发灰或过于拥挤。
  - 让各主页面的配色语言一致，保持同一套视觉语法。
- **完成信号**：
  - 各页面视觉语言统一，且明显呈现“白底 + 蓝紫高亮”的风格。
  - 关键操作按钮、选中态、提示态的颜色有一致规范。
  - 页面整体看起来像一个明亮、干净、专业的桌面级产品界面。
- **验证方式**：
  - 浏览四个主页面，确认白色底板和蓝紫高亮是否贯穿全局。
  - 检查按钮、标签、选中态和提示态是否都使用同一套颜色逻辑。
