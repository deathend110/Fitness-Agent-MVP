# Task 2 RPE 范围修正

- **估时**：15 分钟
- **依赖**：Task 1
- **目标**：确保 RPE 只能在 0-10 之间输入和保存。
- **主要文件**：
  - `src/components/ExerciseEditor.jsx`
  - `src/utils/exerciseForm.js`
  - `tests/`
- **主要操作**：
  - 给 RPE 增加范围校验。
  - 超出范围时给出明确提示。
  - 保存前阻止非法值进入计划数据。
- **完成信号**：
  - RPE 大于 10 或小于 0 时不能被保存。
  - 页面会明确提示输入范围错误。
- **验证方式**：
  - 输入 `11`、`-1`、`10`、`0` 分别验证保存表现。
