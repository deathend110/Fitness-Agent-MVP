# Task 3 每日训练类型开放输入

- **估时**：20 分钟
- **依赖**：Task 2
- **目标**：让每日训练类型不再只依赖固定枚举，支持用户手动填写。
- **主要文件**：
  - `src/components/PlanDayCard.jsx`
  - `src/tabs/PlanTab.jsx`
  - `src/utils/weeklyPlan.js`
- **主要操作**：
  - 调整训练类型输入方式。
  - 保留已有训练日类型的默认值。
  - 让用户可以直接输入自己的训练分类。
- **完成信号**：
  - 训练类型可按自定义内容保存。
  - 刷新页面后仍能恢复。
- **验证方式**：
  - 将某天改成自定义类型，刷新后检查是否仍保留。
