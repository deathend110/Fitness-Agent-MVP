# V2.1 开发完成总结

## 完成范围

- 已按 `docs/coach-ui-design-doc.md` 与 `coach-ui-mockup.html` 重写 AI 教练页。
- AI 教练页已切换为“历史侧栏 + 主聊天区 + 固定底部输入区”的新结构。
- `AppShell` 已为 coach 页提供沉浸式承载模式，外层不再给 coach 页额外滚动和 padding。
- `CoachTab` 已只保留状态协调、发送链路、流式回退、建议采纳和新建对话逻辑。
- 旧的过渡组件已从主流程移除：
  - `CoachConversationPanel.jsx`
  - `CoachSidebar.jsx`
  - `CoachComposer.jsx`
  - `CoachMessageBubble.jsx`

## 关键结果

- 顶部已移除“上下文”按钮，只保留单一聊天视图。
- 空状态支持建议问题点击回填草稿。
- AI 返回结构化建议时，会在对应 AI 消息下方渲染采纳卡片。
- 采纳卡片展示模型与原始 suggestion 已明确分离，采纳计划时继续传递原始 suggestion。
- 历史侧栏使用假多会话展示模型，但历史项 id 已保持稳定，不会因新增消息漂移。

## 验证结果

- 定向测试通过：
  - `tests/coachView.test.js`
  - `tests/coachChat.test.js`
  - `tests/coachGuard.test.js`
  - `tests/adoptCard.test.js`
  - `tests/appShellConfig.test.js`
- `npm run build` 通过。
- 旧组件引用检查已通过，`src` 中不再引用旧 AI 教练页组件。
