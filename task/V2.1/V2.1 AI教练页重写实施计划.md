# V2.1 AI教练页重写实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development（推荐）或 superpowers:executing-plans 按任务执行。步骤使用 checkbox 语法追踪。

**Goal:** 严格按照 `docs/coach-ui-design-doc.md` 与 `coach-ui-mockup.html`，完成 AI 教练页的结构级重写，并移除旧版过渡实现。

**Architecture:** 本次实现先调整 `AppShell` 的 coach 承载模式，建立正确的高度链和唯一滚动区，再以全新的 `src/components/coach/` 组件组重建聊天页。`CoachTab` 只保留状态和事件协调，`AdoptCard` 保留业务接口但改成新视觉风格。

**Tech Stack:** React 19、Vite、Tailwind CSS、Node 内置测试、localStorage 既有状态链路
